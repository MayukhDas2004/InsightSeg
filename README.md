# InsightSeg

AI-Based Brain Tumor Segmentation using BraTS 2021 Dataset and U-Net Architecture.

## Team Members

1. Team Member 1 - AI Model Development and Training
2. Team Member 2 - Infrastructure, Deployment, Version Control
3. Team Member 3 - Visualization and Evaluation
4. Team Member 4 - Documentation and Literature Review

## Project Goal

Develop an AI system capable of automatically segmenting brain tumors from MRI scans using the BraTS 2021 dataset.

## Dataset

* BraTS 2021
* 1251 Patients
* MRI Modalities:

  * FLAIR
  * T1
  * T1CE
  * T2

## Current Progress

* Dataset Builder Completed
